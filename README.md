# Admin-Dashboard
A practice project with HTML and CSS, creating the admin dashboard layout from the provided design(https://raw.githubusercontent.com/TheOdinProject/curriculum/43cc6ab69fdfbef40d431a65677d2144668930ac/intermediate_html_css/grid/project_admin_dashboard/imgs/dashboard-project.png). Main idea was to practice using CSS Grid.

What I Learned

Using CSS Grid for overall layout (sidebar, header, main content)

Using Flexbox for components like cards, header items, and lists

When to use fr vs auto in grid layouts

Debugging spacing and alignment issues in Grid
